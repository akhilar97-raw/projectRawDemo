
## 🚀 Features

- 🌟 Attractive landing page with background image (radical brown theme)
- 🍛 Menu page displaying food items with images and descriptions
- 🎨 Clean UI using pure CSS (no frameworks) and normal html
- 📱 Responsive layout using flexbox
- 🔗 Navigation between pages

## 🖥️ Pages Overview

### 🏠 Landing Page (`index.html`)
- Welcome message
- Background hero image
- "View Menu" button to navigate to menu page

### 🍴 Menu Page (`foodweb.html`)
Displays food items such as:
- Biryani
- Butter Naan
- Veg Dum Biryani
- Mango Lassi
- Rasmalai
- Gulab Jamun

Each item includes:
- Image
- Title
- Description

## ▶️ How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Click on **"View Menu"** to explore the menu page.
